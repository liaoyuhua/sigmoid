**Sigmoid** is a lightweight deep learning framework with JAX as backend.

## Main Feature
* Network Module
* Loss Function
* Optimizer
* Trainer

## Installation
```bash
pip install sigmoid
```

## Usage
```python
import sigmoid as sg
```
